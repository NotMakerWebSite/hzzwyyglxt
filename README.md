### 三、功能描述
系统分为（身份）：参展商、管理员。

**参展商功能：**
1. 登录
2. 注册
3. 活动控制台（活动预约）
4. 展位控制台（展位预约）
5. 活动物品租赁
6. 我的租赁
7. 留言区
8. 公告查看
9. 安全退出

**管理员功能：**
1. 登录
2. 参展商管理
2. 活动控制台（活动管理与审批）
3. 展位一览
4. 物品管理
5. 租赁申请
6. 会展留言区
7. 公告发布
8. 安全退出
